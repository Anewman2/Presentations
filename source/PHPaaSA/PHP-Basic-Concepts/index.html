PHP Tags

Opening and Closing tags

  1. <?php and ?> Always Available
  2. <script language="php"> and </script> Always Available
  3. <? and ?> Short Style
  4. <?= and ?> Echo - Short Style
  5. <% and %> ASP Style
  6. <%= and %> Echo - ASP Style

The opening tag tells the interpreter to turn on

The closing tag tells the interpreter to turn off

  • Tag Sets 1 and 2 are always available
    • Other tag sets should be avoided when developing for redistribution
    • Set 1 is most common and recommended
  • Tag Sets 3 and 4 are Short style – Requires a configuration change
    • Configuration file (php.ini) – short_open_tag
    • Compiled with --enable-short-tags
    • Since ver 5.4 parser recognizes short tags regardless of configuration
  • Tag Sets 5 and 6 are ASP style – Requires a configuration change
    • Configuration file (php.ini) – asp_tags
  • Since ver 5.3 parser allows an opening tag (minimum of one white space character) by itself in a file
  • A Close tag at the end of the file is optional
    • Helpful so unwanted white space does not occur
